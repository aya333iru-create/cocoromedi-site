# CocoroMedi Site

## Overview
CocoroMedi is a web project aimed at providing mental health support for healthcare professionals. This project includes a main HTML file, styles, scripts, and components to create a cohesive and interactive user experience.

## Project Structure
```
cocoromedi-site
├── src
│   ├── index.html          # Main entry point for the website
│   ├── styles
│   │   └── style.css       # CSS styles for the website
│   ├── scripts
│   │   └── main.js         # JavaScript for interactivity
│   ├── components
│   │   ├── nav.html        # Navigation component
│   │   └── footer.html     # Footer component
│   └── data
│       └── site.json       # JSON data for dynamic content
├── public
│   └── fonts               # Directory for font files
├── .vscode
│   └── settings.json       # VS Code specific settings
├── package.json            # npm configuration file
└── README.md               # Project documentation
```

## Setup Instructions
1. Clone the repository:
   ```
   git clone <repository-url>
   ```
2. Navigate to the project directory:
   ```
   cd cocoromedi-site
   ```
3. Install dependencies:
   ```
   npm install
   ```
4. Open the project in your preferred code editor.

## Usage
- Open `src/index.html` in a web browser to view the website.
- Modify styles in `src/styles/style.css` to change the appearance.
- Update `src/scripts/main.js` for any JavaScript functionality.
- Use the components in `src/components/` to maintain a consistent layout.
